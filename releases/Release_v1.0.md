# Release v1.0 - Personal Expense Tracker

**Release Date:** 01/02/2026  
**Baseline Reference:** BL2

## Overview
This is the **initial working system** of the Personal Expense Tracker project.  
It includes the core functionalities implemented after BL2:

- Login / Authentication page
- Main Dashboard displaying expenses
- Add / Edit / Delete expense functionality
- Basic JSON database for storing data
- Navigation via Navbar
- Basic UI components: ExpenseList, ExpenseItem
- TypeScript support and initial configurations

## Included Files / Folders
- `src/` : Source code (components, pages, App.tsx, index.tsx, types.ts)
- `data/` : JSON database (users / expenses)
- `docs/` : SCM documents (SCMP, CI Register, Baseline Records)
- `tests/` : Placeholder for future unit / integration tests
- `releases/` : This release notes file
- `README.md` : Project overview
